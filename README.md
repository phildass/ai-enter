## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure environment variables:**
   ```bash
   cp .env.example .env
   ```
   Edit `.env` and add your Razorpay and Supabase credentials.

3. **Run development server:**
   ```bash
   npm run dev
   ```
   Visit http://localhost:3000

4. **Build for production:**
   ```bash
   npm run build
   npm start
   ```
